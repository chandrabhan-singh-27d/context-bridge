# context-bridge

MCP (Model Context Protocol) server giving any AI assistant structured access to a GitHub repository — issues, PRs, diffs, CI, commits — plus a companion chat UI demonstrating it.

Built on **Bun + Hono**. Sub-50ms cold start. Single-binary distribution.

> 🚧 Bootstrap in progress. See [AI_PROJECTS.md](https://github.com/chandrabhan-singh-27d) for the full spec.
